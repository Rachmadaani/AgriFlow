**AgriFlow** adalah aplikasi berbasis website untuk memonitoring usaha pertanian skala kecil. Dibangun dengan Laravel dan dirancang khusus untuk membantu petani mencatat panen harian, mengelola keuangan, serta menganalisis keuntungan usaha tani.

![Dashboard AgriFlow](https://via.placeholder.com/800x400?text=AgriFlow+Dashboard)

---

## ✨ Fitur Unggulan

### 📊 Dashboard Interaktif
- Ringkasan total panen minggu ini
- Grafik panen 7 hari terakhir
- Rekapitulasi pengeluaran per bulan
- Estimasi keuntungan usaha

### 🌱 Manajemen Tanaman
- Tambah/edit/hapus jenis tanaman (Cabai, Timun, dll)
- Atur harga default per kilogram
- Catat tanggal tanam

### 📝 Input Panen Harian
- Catat hasil panen per hari
- Input berat (kg) dan harga jual
- Otomatis hitung pendapatan

### 💰 Manajemen Pengeluaran
- Catat biaya operasional (bibit, pupuk, pestisida, transportasi)
- Kategorisasi pengeluaran
- Ringkasan per bulan

### 🤖 AI Chatbot (DeepSeek)
- Konsultasi masalah pertanian
- Tips budidaya cabai dan timun
- Informasi harga dan hama

### 📱 Responsive Design
- Tampilan optimal di Desktop, Tablet, dan Mobile
- Mudah digunakan oleh petani pemula
